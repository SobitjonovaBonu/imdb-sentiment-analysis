# IMDB Sentiment Analysis (RNN vs LSTM)

## 📌 Project Overview
This project compares RNN and LSTM models on IMDB movie reviews dataset.

## 🎯 Task
Classify movie reviews as positive or negative.

## ⚙️ Models Used
- RNN
- LSTM

## 📊 Results

| Model | Accuracy |
|------|----------|
| RNN  | 76.6%    |
| LSTM | 83.8%    |

## 🧠 Conclusion
LSTM performs better than RNN because it can capture long-term dependencies using memory cells and gates.

## 🛠️ Tech Stack
- Python
- PyTorch
- TensorFlow (for dataset)
